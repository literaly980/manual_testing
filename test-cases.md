# Тест-кейсы — Swag Labs (saucedemo.com)

Окружение: Chrome (последняя версия), Windows 11, https://www.saucedemo.com/
Пароль для всех пользователей: `secret_sauce`

---

## Чек-лист (быстрая проверка покрытия)

**Авторизация**
- [ ] Вход с верными данными (standard_user)
- [ ] Вход с неверным паролем
- [ ] Вход заблокированного пользователя
- [ ] Вход с пустым логином / пустым паролем
- [ ] Выход из аккаунта (logout)

**Каталог товаров**
- [ ] Отображаются 6 товаров (название, цена, картинка, кнопка)
- [ ] Сортировка по имени (A–Z, Z–A)
- [ ] Сортировка по цене (по возрастанию / убыванию)
- [ ] Открытие карточки товара

**Корзина**
- [ ] Добавление товара (счётчик +1)
- [ ] Добавление нескольких товаров
- [ ] Удаление товара из корзины

**Оформление заказа**
- [ ] Полный сценарий до «Thank you for your order!»
- [ ] Валидация: пустые поля имени/фамилии/индекса

---

## Детальные тест-кейсы

### TC-01. Успешный вход
**Приоритет:** High
**Предусловие:** открыта страница https://www.saucedemo.com/
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Ввести логин `standard_user` | Поле заполнено |
| 2 | Ввести пароль `secret_sauce` | Поле заполнено |
| 3 | Нажать «Login» | Открывается страница каталога (URL `/inventory.html`), виден список товаров |

### TC-02. Вход с неверным паролем
**Приоритет:** High
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Логин `standard_user`, пароль `wrong_pass` | Поля заполнены |
| 2 | Нажать «Login» | Показана ошибка: «Username and password do not match any user in this service» |

### TC-03. Вход заблокированного пользователя
**Приоритет:** High
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Логин `locked_out_user`, пароль `secret_sauce` | Поля заполнены |
| 2 | Нажать «Login» | Показана ошибка: «Sorry, this user has been locked out.» |

### TC-04. Вход с пустым логином
**Приоритет:** Medium
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Оставить логин пустым, пароль `secret_sauce` | — |
| 2 | Нажать «Login» | Показана ошибка: «Username is required» |

### TC-05. Вход с пустым паролем
**Приоритет:** Medium
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Логин `standard_user`, пароль пустой | — |
| 2 | Нажать «Login» | Показана ошибка: «Password is required» |

### TC-06. Добавление товара в корзину
**Приоритет:** High
**Предусловие:** выполнен вход под `standard_user`
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Нажать «Add to cart» у любого товара | Кнопка меняется на «Remove» |
| 2 | Посмотреть значок корзины | На значке отображается «1» |

### TC-07. Удаление товара из корзины
**Приоритет:** Medium
**Предусловие:** в корзине 1 товар
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть корзину | Товар отображается в списке |
| 2 | Нажать «Remove» | Товар удалён, счётчик корзины пуст |

### TC-08. Сортировка по цене (по возрастанию)
**Приоритет:** Medium
**Предусловие:** открыт каталог
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Выбрать в сортировке «Price (low to high)» | Товары отсортированы по возрастанию цены (первый — самый дешёвый) |

### TC-09. Оформление заказа (полный сценарий)
**Приоритет:** High
**Предусловие:** в корзине ≥1 товар, вход под `standard_user`
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть корзину → «Checkout» | Открыта форма данных покупателя |
| 2 | Ввести First Name, Last Name, Zip → «Continue» | Открыт обзор заказа (Overview) |
| 3 | Нажать «Finish» | Сообщение «Thank you for your order!» |

### TC-10. Оформление заказа с пустыми полями
**Приоритет:** Medium
**Предусловие:** открыта форма Checkout
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Оставить поля пустыми → «Continue» | Показана ошибка «First Name is required» |

### TC-11. Выход из аккаунта
**Приоритет:** Low
| # | Шаг | Ожидаемый результат |
|---|-----|---------------------|
| 1 | Открыть меню (бургер) → «Logout» | Возврат на страницу входа |
