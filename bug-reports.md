# Баг-репорты — Swag Labs (saucedemo.com)

Окружение: Chrome (последняя версия), Windows 11, https://www.saucedemo.com/

---

## BUG-01. У пользователя `problem_user` все товары с одинаковой картинкой

**Severity:** Major
**Priority:** High
**Окружение:** Chrome, Windows 11

**Шаги воспроизведения:**
1. Открыть https://www.saucedemo.com/
2. Войти под `problem_user` / `secret_sauce`
3. Посмотреть изображения товаров в каталоге

**Ожидаемый результат:** у каждого товара своя картинка.
**Фактический результат:** у всех товаров одинаковое (неверное) изображение.

---

## BUG-02. У пользователя `performance_glitch_user` каталог грузится с большой задержкой

**Severity:** Minor
**Priority:** Medium
**Окружение:** Chrome, Windows 11

**Шаги воспроизведения:**
1. Открыть https://www.saucedemo.com/
2. Войти под `performance_glitch_user` / `secret_sauce`
3. Замерить время загрузки страницы каталога

**Ожидаемый результат:** страница каталога загружается без заметной задержки
(как у `standard_user`).
**Фактический результат:** страница каталога загружается заметно дольше
(несколько секунд).

## BUG-03. `problem_user`: при оформлении заказа не сохраняется поле «Last Name»

**Severity:** Major
**Priority:** High
**Окружение:** Chrome, Windows 11

**Шаги воспроизведения:**
1. Войти под `problem_user` / `secret_sauce`
2. Добавить товар в корзину → открыть корзину → «Checkout»
3. Заполнить First Name, Last Name, Zip
4. Нажать «Continue»

**Ожидаемый результат:** форма принимает данные, открывается обзор заказа.
**Фактический результат:** поле «Last Name» работает некорректно (ввод не
принимается / появляется ошибка), перейти к обзору заказа не удаётся.

---

## Шкала важности (для справки)

- **Priority** (как срочно чинить): High → Medium → Low
